定义在 bridge.js:298

可用版本 v5.2

将log写入到native的日志界面，该函数已移动到CtripUtil类，此处只做兼容。具体参考CtripUtil.app_log()函数

参数:

  • log String

    需要打印打log

  • result String

    上一句log执行的结果，可以为空,打印的时候会自动换行，加入时间

举例:

CtripUtil.app_log("execute script xxxxx", "result for script is oooooo");
