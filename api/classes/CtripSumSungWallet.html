app_check_ticket_in_samsung_wallet

(
  • ticketID
)

定义在 bridge.js:2065

可用版本 v5.3.2

检查ticket是否在三星钱包app中

参数:

  • ticketID String

    ticket的ID，服务器返回

举例:

CtripSumSungWallet.app_check_ticket_in_samsung_wallet("ID123333");

  //调用之后会收到
   var json_obj = {
       tagname : "check_ticket_in_samsung_wallet",
       param : {
           insInSamSungWallet: false, //true
       }
   }

   app.callback(json_obj);
